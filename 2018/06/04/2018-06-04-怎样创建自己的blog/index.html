手把手搭建博客

发表于 2018-06-04

一、前期工作

（一）创建Github 域名和空间

  1. 注册github账号，设定username，之后设立的域名就是username.github.io
  2. 创建新的仓库(repository) ，username 一定与前面的Owner一致

（二） 安装git，nodejs，hexo

  1. 通过homebrew安装git
  1. 安装Nodejs

  2. 首先安装nvm管理器

​ 再重启终端，安装 Node.js。$ nvm install 4

  1. 安装博客框架hexo

    sudo npm install hexo-cli -g

二、搭建博客

（一）创建博客

  1. 使用hexo创建博客，并在本地生成名为username.github.io的博客目录。

hexo init *username*.github.io

（二）下载主题

以最受欢迎的*next*主题为例，更多主题见可从[hexo的官网](https://hexo.io/themes/)获取，也可参考[知乎《有哪些好看的 Hexo 主题？》](有哪些好看的 Hexo 主题？)。

  1. 定位到生成的博客目录cd *username*.github.io

  2. 获取选定主题next在github上的托管地址 https://github.com/iissnan/hexo-theme-next

    将其克隆至username.github.io文件夹下的themes文件夹中

    git clone https://github.com/iissnan/hexo-theme-next themes/next

（三）配置站点：

​ 注意hexo和next都有配置文件，且命名相同。

​ 前者为username.github.io_config.yml，被称为站点配置文件，主要参考hexo配置文档；

​ 后者在username.github.io\themes\next_config.yml，被称为主题配置文件，主要参考next配置文档。

​ 修改下列选项，注意:

​ language填写zh-CN，而不是官方文档中的zh-Hans因为后续会要利用\next\languages\zh-CN.yml文件对博客进行语言切换。

（四）配置主题

​ 每篇博客都可以设置几个标签，利用标签页面就能够分标签浏览。以添加「标签」页面为例，主要讲一下如何新建子页面。其他配置参考官方文档即可。

  • 首先，定位到本地hexo目录。

    cd *username*.github.io

  • 在username.github.io\sources下生成tags新文件夹。

    hexo new page tags。

  • 修改username.github.io\sources\tag\index.md文件。

    修改title: 标签

    增加type: "tags"

  • 同理可以添加categories等页面。

  • 在主题配置文件\themes\next_config.yml中找到menu菜单，将新建的tags和categories等页面开启。

  1. 注册百度统计
  2. 注册畅言

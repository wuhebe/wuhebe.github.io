【效率工具】一枝红杏出墙来:自建ss账号

发表于 2018-07-14 | 分类于 效率工具 | | 阅读次数：

目标

科学冲浪

工具

  1. visa双币种信用卡
  2. 电脑：mac pro
  3. 耐心

步骤

创建支付渠道——购买国外服务器——mac连接——配置ss账号

创建支付渠道

  1. paypal官网
  2. 按步骤注册并验证邮箱，绑定信用卡。初次绑定时，会扣取1美元测试，大概半个小时之后会发起退款。
  3. paypal注册会有不定时优惠活动，之前新用户注册可以申请领取10美元代金券。需要致电paypal客服人员主动申请。2018年7月15日我注册时，这一活动已经结束，但客服态度极佳，鉴于我电话咨询，主动帮助申请5美元注册红包，后来实际到账10美元。服务体验极佳。

购买国外服务器

  1. Vultr官网

  2. 关注Vultr官网的活动信息，按步骤注册。2018年7月15日我注册时，有两项活动：

    1. 新用户注册送25美元代金券
    2. 充值100美元送100美元

    轻度用户果断选择第一种方式。

  3. Vultr绑定Paypal账号。需要用Paypal向Vultr充值10美元。刚好使用上述10美元代金券，账户余额35美元。

  4. 购买虚拟服务器。

    Vulrt——Servers——右侧的＋号，deploy一个新的VPS。

    Server Location选择Tokyo（或Los Angeles）

    Server type选择64 bit OS 下面的centos

    Server Size选月付5美元或者月付10美元

    点击右下角的Deploy Now生成，之后跳转到管理页面（Servers），当显示绿色的running时，该完成部署。可获取VPS的IP和密码信息。
